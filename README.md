Задание 1.1 — Класс Point

Описание
Создан класс Point, описывающий точку на двумерной плоскости с координатами X и Y.
Реализован метод toString() для вывода в формате {X;Y}.

Логика решения

Координаты X и Y представлены целыми числами.

Поля объявлены как private.

Точка создаётся только через конструктор с двумя параметрами.

Метод toString() возвращает строку {X;Y}.

Пример

Ввод:  new Point(3, 5)
Вывод: {3;5}

Задание 2.1 — Класс Line

Описание
Создан класс Line, описывающий линию на плоскости, заданную начальной и конечной точками.

Логика решения

Начало и конец линии представлены объектами класса Point.

Используется конструктор с двумя точками.

Метод toString() возвращает строку
Линия от {X1;Y1} до {X2;Y2}.

Изменение координат выполняется заменой объектов Point.

Пример

Ввод:  Линия с началом {1;3} и концом {23;8}
Вывод: Линия от {1;3} до {23;8}

Задание 3.1 — Класс Student

Описание
Создан класс Student, описывающий студента по имени и массиву оценок.

Логика решения

Имя хранится в виде строки.

Оценки хранятся в массиве целых чисел.

toString() возвращает строку
Имя: [оценка1, оценка2, …].

При копировании массива копируется ссылка, а не данные.

Пример

Вася: [3, 4, 5]
Петя копирует оценки Васи

После изменения оценки:

Вася: [5, 4, 5]
Петя: [5, 4, 5]

Задание 5.6 — Класс StudentV3 (Студент-отличник)

Описание
Студент с определением среднего балла и признака отличника.

Логика решения

Средний балл считается как среднее арифметическое.

Если оценок нет — средний балл равен 0.

Отличник — есть хотя бы одна оценка и все оценки равны 5.

Пример

Вася: средний = 4.0, отличник = false
Петя: средний = 5.0, отличник = true
