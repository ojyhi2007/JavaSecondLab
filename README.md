## Задание 1.1 — Класс Point

### Описание
Создан класс Point, описывающий точку на двумерной плоскости с координатами X и Y.

### Логика решения
- Координаты X и Y представлены целыми числами
- Поля объявлены как private
- Создание точки возможно только через конструктор с двумя параметрами
- Метод toString() возвращает строку в формате {X;Y}

### Пример
Ввод: new Point(3, 5)  
Вывод: {3;5}
